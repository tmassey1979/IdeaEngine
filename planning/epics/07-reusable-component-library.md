# [Epic] REUSABLE COMPONENT LIBRARY

Source section: `codex/sections/07-reusable-component-library.md`

## User Story

As a platform operator, I want a reusable component library, so that new projects can be assembled faster with less duplicated code.

## Summary

Dragon Idea Engine should avoid repeatedly generating the same infrastructure code across projects.

Instead, it maintains a **Reusable Component Library** consisting of standardized modules and shared services.
