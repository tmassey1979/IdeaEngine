# Codex: AUTONOMOUS_IDEA_DISCOVERY_SYSTEM

Source section: `codex/sections/19-autonomous-idea-discovery-system.md`

## Summary

The Autonomous Idea Discovery System enables Dragon Idea Engine to proactively discover, generate, and evaluate new ideas without requiring direct user input.
Instead of relying only on submitted ideas, the system continuously scans multiple information sources to detect:
```text
unsolved problems
emerging technology trends
inefficiencies in existing tools
new business opportunities
scientific and engineering gaps
```
These discoveries are then submitted into the Idea Evaluation Pipeline.
