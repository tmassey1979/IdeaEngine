Example workflow for building a project:

``` id="orch-5"
Idea Approved
      ↓
Architecture Design
      ↓
Backend Development
      ↓
Frontend Development
      ↓
Testing
      ↓
Deployment
      ↓
Documentation
```

Each step triggers a new task in the system.
