### Source Excerpt

When improving an agent, the system may create experimental variants.

Example:

```text
Agent_V1  (current version)
Agent_V2A (optimized prompts)
Agent_V2B (different reasoning strategy)
Agent_V2C (different model configuration)
```

These variants are tested against benchmark tasks.

The best performing variant replaces the original agent.
