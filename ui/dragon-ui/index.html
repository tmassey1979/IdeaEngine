Command the forge. Track every story, failure, and recovery chain.

Dragon is an autonomous software delivery platform that turns ideas into tracked execution. It reads GitHub work, runs implementation loops, keeps workflow state in sync, manages recovery when delivery stalls, and gives operators one place to submit ideas, monitor progress, and guide active projects. An embedded ethics agent helps enforce safety boundaries, keeping autonomy aligned with operator intent, review gates, and accountable decision-making.
